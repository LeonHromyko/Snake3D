# Fabros Snake3D Test Task #

Make a Snake game in 3D. Field 15x15x15.
Two snakes move automatically on the field. Initial length 3 cells.
Move along the cells in random directions (up, down, left, right, forward).
Apples are generated on the field and when it is eaten the snake size increases by 1.
If an apple exists in some direction from the snake moves to it.
The snake should avoid colliding with itself or with another snake. In a collision dies.
